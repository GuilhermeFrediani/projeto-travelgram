# Travelgram

![Capa do Projeto](./assets/projeto%20travelgram.png)

## O que é o projeto?
O Travelgram é uma página de perfil de viagens estruturada para exibir informações pessoais e galerias de fotos de forma atrativa.

## O que eu aprendi
Neste projeto, aprendi a estruturar melhor o meu CSS. Ao invés de criar um único arquivo gigantesco (como um genérico `style.css`), criei uma pasta chamada `styles` e, dentro dela, separei as regras em arquivos `.css` específicos (como `header.css`, `footer.css`, `global.css`, `main.css`, etc.).

Isso centraliza as estilizações de forma muito mais organizada e estruturada. Descobri que essa prática é muito importante para a manutenção do código, pois no futuro será muito mais fácil melhorar arquivos específicos, identificar onde pode estar um erro e corrigi-lo sem o risco de danificar ou quebrar um único arquivo CSS gigante.
